# Titanic-Survival-Prediction

Project Overview

This project uses Machine Learning to predict whether a passenger survived the Titanic disaster.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

Machine Learning Model

- Random Forest Classifier

Dataset

- train.csv
- test.csv

Features Used

- Passenger Class (Pclass)
- Sex
- Age
- Fare

Output

The model generates a CSV file containing survival predictions for test passengers.

Learning Outcomes

- Data Preprocessing
- Feature Engineering
- Classification Models
- Model Evaluation
- Prediction Generation
